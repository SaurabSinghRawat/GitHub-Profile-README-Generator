Add-ons

Add-ons are optional features you can enable to customize and enhance your README file. Think of them as powerful extras you can toggle on/off based on what you need.

The GitHub Profile README Generator integrates several powerful open-source add-ons developed by talented developers from the GitHub community. We’re incredibly grateful to the developers behind them.



README Typing SVG

"⚡ Dynamically generated, customizable SVG that gives the appearance of typing and deleting text for use on your profile page, repositories, or website."

Developed by Jonah Lawrence.



LeetCode Badges

"🏅 A showcase of your LeetCode achievement badges — easy to embed, with multiple themes and animated option."

Developed by KevzPeter.



LeetCode Stats Card

"Show your dynamically generated LeetCode stats on your GitHub profile or your website!"

Developed by JacobLinCool.



GitHub README Stats

"Dynamically generated stats for your github readmes."

Github Stat Card

Top Languages Card

Developed by Anurag Hazra.



Github Readme Streak Stats

"🔥 Stay motivated and show off your contribution streak! 🌟 Display your total contributions, current streak, and longest streak on your GitHub profile README."

Developed by Jonah Lawrence.



Github Profile Views Counter

"It counts how many times your GitHub profile has been viewed. Free cloud micro-service."

Developed by Anton Komarev.



Github Profile Trophies

"🏆 Add dynamically generated GitHub Stat Trophies on your readme."

Developed by ryo-ma.

Icons & Images

We are grateful to the talented creators who provide the icons and images that enhance the visual appeal and functionality of the GitHub Profile README Generator. Their contributions make our tool more user-friendly and engaging.

Icons from ICONS8

For various brand icons, especially for programming languages, technologies, and services, we utilize icons from Icons8.

Icons8 provides a vast library of vector icons and social logos, making it easy to integrate scalable and customizable visual elements.

Thank you, Icons8, for your excellent resource!

Icons from Vector Logo Zone

Many of the common icons used throughout this generator, such as some Programming languages icons, and general interface icons, are sourced from Vector Logo Zone.

Vector Logo Zone offers free SVG icons for popular brands, perfect for consistent branding across different platforms.

Our sincere thanks to the Vector Logo Zone team!

Some Others..

Our appreciation also goes to Iconify.Design and Font Awesome. Finally, we acknowledge OpenResty (official website).

We appreciate these platforms for making their brand assets available for use.
